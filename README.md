# mystery-box

A curated collection of data engineering and AI tips, tricks, and prompts — distilled from real-world work and shared on [LinkedIn](https://www.linkedin.com/in/).

## What's here

Practical, production-oriented content across:

- **AI & LLM** — prompts, workflows, Claude/GPT setups, agent patterns
- **SQL & data modeling** — Snowflake, BigQuery, PostgreSQL patterns worth keeping
- **BI & analytics** — Tableau, dbt, Dataform, KNIME
- **Python** — data pipeline snippets, automation, tooling

No tutorials. No padding. If it's here, it earned its place.

## Usage

Browse by folder or search for a keyword. Everything is standalone — no dependencies between files unless noted.

## Context

I'm a senior data analyst and BI engineer running a solo consultancy in Poland. This repo is a living document — updated as I post, discover, or build something worth sharing.
